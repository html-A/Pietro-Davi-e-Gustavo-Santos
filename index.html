Exercícios de JavaScript

Exercício 1

Neste exercício, o programa irá pedir para o usuário digitar um número inteiro, e o programa irá identificar se
o número digitado é negativo, neutro ou positivo


Exercício 2

Neste exercício, o programa irá pedir para o usuário digitar sua idade,a partir daí, o programa irá determinar se
você é criança, adolecente, adulto, ou idoso com base na sua idade


Exercício 3

Neste exercício, o programa irá pedir para o usuário digitar um número de 1-7, a partir dai,o programa irá vincular este número a um dia da semana e irá mostrar para o usuário
exemplo: 1 = domingo


Exercício 4

Neste exercício,o programa irá altomaticamente, por conta de um laço FOR,fazer uma sequência de 1 a 10, que vai aparecer na tela do usuário


Exercício 5

neste exercício, o programa irá pedir para o usuário digitar números inteiros, até o usuário digitar o número 0, a partir dai, o programa irá somar todos os números altomaticamente
e vai mostrar o resultado na tela


Exercício 6

neste exercício, o programador irá colocar um número inteiro surpresa entre 1 e 10, e o programa vai pedir para o usuário adivinhar, se o usuário conseguir, vai receber um parabéns


Exercício 7

neste exercício, o programa vai pedir para o usuário digitar números inteiros 5 vezes, a partir dai, o programa vai mostrar o menor e o maior número


Exercício 8

Neste exercício, o programa vai pedir ao usuário uma quantidade de valores, daí o prorama vai coletar esses valores, irá calcular a média, e vai exibr a média dos valores na tela


Exercício 9

neste exercício, o programa vai pedir para o usuário digitar um número de 1 a 9, daí o programa vai comparar o número com o número 9
contando quantos números faltam para chegar do número escolhido pelo usuário número para o número 9


Exercício 10

neste exercício, o programa vai dar uma lista de palavras, e vai pedir para o usuário escolher uma digitando essa palavra
daí o programa vai mostrar a quantidade de vezes essa palavra aparece no programa
